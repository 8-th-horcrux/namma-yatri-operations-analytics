# namma-yatri-operations-analytics
Operational analytics dashboard built using Power BI and DAX to analyze ride demand, revenue, cancellations, customer behavior, and service efficiency for a ride-hailing platform. Following highlights include from namma yatri dataset analysis:

2,161 ride requests analyzed
983 completed rides
Ride funnel analysis
Cancellation analysis
Revenue analysis
Zone-level performance analysis
Demand forecasting by time period
Payment behavior analysis
